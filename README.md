# Invoice System - Multi User Role

Aplikasi invoice modern dengan sistem user role (Admin, Gudang, Sales, Customer, Finance) menggunakan **LocalStorage** tanpa database MySQL.

## 🚀 Demo Credentials

## ✨ Fitur Per Role

### **Admin** (Full Access)
- Dashboard lengkap
- Kelola semua user
- Lihat semua invoice
- Kelola stok produk

### **Gudang** 
- Monitor stok real-time
- Lihat invoice pending
- Update stok otomatis

### **Sales**
- Buat invoice baru
- Lihat invoice miliknya
- Pilih produk dari stok

### **Customer**
- Lihat invoice pribadi
- Status invoice real-time

### **Finance**
- Approve invoice
- Monitor invoice pending & approved
- Lihat total nilai invoice

## 📱 Responsive Design
- Mobile-first approach
- Print-friendly invoice
- Dark sidebar navigation

## 🛠 Tech Stack
- Vanilla JavaScript (No Framework)
- LocalStorage (No Database)
- CSS Grid & Flexbox
- Font Awesome Icons

## 🚀 Deploy ke GitHub Pages
1. Upload semua file ke repository GitHub
2. Settings → Pages → Deploy from branch `main`
3. Akses `https://username.github.io/repo-name`

## 📊 Data Persisten
Semua data tersimpan di **LocalStorage** browser:
- Users, Invoices, Products, Stok
- Data aman selama browser tidak di-clear

## 🎨 Customization
Edit `css/style.css` untuk tema custom
Tambah role baru di `js/dashboard.js`
